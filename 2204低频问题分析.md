# 2204 电机低频不转 vs 高频能转 — 物理分析

## 先认清楚电机的"身份证"参数

这些缩写贯穿整个 FOC 学习，逐个解释清楚：

| 参数 | 值 | 全称 | 大白话解释 |
|------|-----|------|-----------|
| **Np** | 7 | Number of Pole Pairs（极对数） | 转子上有几对磁铁。Np=7 意味着转子一圈有 14 个磁极。电频率 = 机械频率 × Np |
| **Rs** | 0.112 Ω | Stator Resistance（定子电阻） | 每相绕组铜线的电阻。0.112Ω 非常小，意味着给一点电压就能流出大电流 |
| **Ls** | 9 μH | Stator Inductance（定子电感） | 绕组的电感值。9μH（微亨）极小，普通电机通常在毫亨（mH）级别。电感是"电流变化的阻力" |
| **Flux** | 0.00034 Wb | Permanent Magnet Flux Linkage（永磁磁链） | 转子上磁铁有多强。单位 Wb（韦伯）。这个值乘以转速就是反电动势的大小 |
| **KT** | 0.00357 Nm/A | Torque Constant（转矩常数） | 每 1 安培电流能产生多大转矩。KT = 1.5 × Np × Flux。你的电机 KT 极小，说明"出力很软" |
| **τe** | **80 μs** | Electrical Time Constant（电气时间常数） | τe = Ls/Rs。电流建立到稳态的 63% 需要的时间。80μs 刚好 ≈ 一个 PWM 周期，电流变化极快 |
| **KV** | 2300 | Velocity Constant（速度常数） | 每 1V 电压电机空载转多快。2300KV 意味着 1V → 2300 RPM，12V → 27600 RPM |

## 单位和前缀速查

| 符号 | 读法 | 含义 | 例 |
|------|------|------|-----|
| μH | 微亨 | 10⁻⁶ 亨 | 9 μH = 0.000009 H |
| mH | 毫亨 | 10⁻³ 亨 | 普通电机 0.1~10 mH |
| Wb | 韦伯 | 磁通量单位 | 0.00034 Wb 是很小的磁链 |
| Nm | 牛·米 | 转矩单位 | 0.014 Nm ≈ 手里捏着一根 14cm 的吸管 |
| rad/s | 弧度每秒 | 角速度单位 | 1 圈/秒 = 6.28 rad/s |
| RPM | 转每分 (Revolutions Per Minute) | 转速常用单位 | 1000 RPM ≈ 105 rad/s |

## 为什么 2Hz/3V 转不起来

### 1. 低频下——电机退化为一段铜线

所谓"低频"是相对电机参数而言的。需要两个关键量：

**感抗 XL**（Inductive Reactance）：
- 电感对交流电的"阻力"
- 公式：XL = ωe × Ls，其中 ωe（读作 omega-e）是电角速度
- 频率越高，感抗越大；频率越低，感抗越小

**电阻 Rs**：铜线的纯电阻，和频率无关。

```
机械频率 fm = 2 Hz（每秒转 2 圈，即 17 RPM）
→ 电频率 fe = Np × fm = 7 × 2 = 14 Hz
→ 电角速度 ωe = 2π × 14 = 88 rad/s

感抗: XL = ωe × Ls = 88 × 0.000009 = 0.00079 Ω
电阻: Rs = 0.112 Ω

XL / Rs = 0.007   ← 感抗不到电阻的 1%！
```

| | 2Hz 机械（17 RPM） | 20Hz 机械（171 RPM） |
|:--|:--:|:--:|
| 电频率（fe） | 14 Hz | 140 Hz |
| 电角速度（ωe） | 88 rad/s | 880 rad/s |
| 感抗（XL） | 0.00079 Ω | 0.0079 Ω |
| 反电动势 (**BEMF**) | 0.03 V | 0.30 V |
| 阻抗谁来主导 | **Rs 绝对主导** | Rs 主导，BEMF 开始参与 |

> **BEMF** = Back ElectroMotive Force（反电动势） = ωe × Flux
> 电机旋转时磁铁划过线圈，线圈自己发电产生的电压。转速越快，BEMF 越大。
> 外加电压必须大于 BEMF 才能"推进"电流。

**结论：2Hz 时 XL 可以忽略不计，电机电气特性 ≈ 一根 0.112Ω 的铜线。**

---

### 2. 电流天然爆炸大——这是'锅'

给定 v_d = 3V，流过 0.112Ω 铜线：

```
理论电流 I = 3 / 0.112 ≈ 27 A
```

27A！你的电机额定才 12A。

实际系统不会让它到 27A——电流限幅会切掉占空比。但限幅一旦介入：

```
你想给 3V → 电流太大了 → 限幅切占空比 → 实际只给 ~1.2V
→ 有效电流降到 10A → 再被进一步限制 → ...（循环压缩）
```

**2Hz 时没有反电动势帮忙吸收电压**：
- BEMF = 0.00034 × 88 = 0.03V，几乎为零
- 3V 全压在 0.112Ω 上
- 20Hz 时 BEMF=0.3V，虽然也很小，但至少消耗了一部分电压

```
可用 q 轴电流（产生转矩的电流）估算:
Iq_max ≈ (V_available - BEMF) / Rs

2Hz:  Iq_max ≈ (1.5 - 0.03) / 0.112 ≈ 13 A
20Hz: Iq_max ≈ (1.5 - 0.30) / 0.112 ≈ 10.7 A
```

看起来 2Hz 理论 Iq 更高。但实际系统中**电流大了必然触发限幅**，限幅后有效电压更少，形成恶性循环。

---

### 3. 转矩打不过齿槽——这是'盖'

**齿槽效应**（Cogging Torque）：
- 永磁转子会天然被"吸"在某些角度位置（磁铁和铁芯齿的对齐位置）
- 每个电机都有，2204 这种小电机尤其明显
- 齿槽峰值力矩通常在 **0.005~0.02 Nm** 量级

"齿槽"就像转子被很多微型凹坑卡着，不使劲拔不出来。

```
可用电磁转矩  Te = KT × Iq

KT = 0.00357 Nm/A（每安培只能产生 0.00357 牛·米）

如果电流限在 4A:  Te = 0.00357 × 4 = 0.0143 Nm
如果限在 2A:      Te = 0.00357 × 2 = 0.0071 Nm
```

如果齿槽峰值 > 0.01 Nm，而能出的转矩只有 0.01 Nm，转子就被卡住——**stall（闷转）**。

> **Stall** = 堵转/闷住。电机通电但转子不动，电流全变成热量。
> 这不是"转子和磁场不同步"，而是根本挣脱不了吸住它的位置。

---

### 4. 为什么 20Hz 反而能转

**惯性**（Inertia）帮你冲过去了。

```
2Hz  = 17 RPM   → 转子每秒钟才转 0.28 圈，每段停留时间长
20Hz = 171 RPM  → 转子每秒转 2.85 圈，有 100 倍的动能
```

类比：
- 走路推过减速带（2Hz）→ 减速带每 10 秒来一次 → 卡住
- 跑步冲过去（20Hz）→ 减速带秒过 → 惯性带着冲过去

**20Hz 时转子动能大，遇到齿槽凹坑更多是靠惯性"蹭"过去的。** 一旦冲过去，下一个换相段立刻接上推力。

## 物理公式速查（附全称对照）

### 阻抗

| 符号 | 全称 | 含义 |
|------|------|------|
| Z | Impedance | 阻抗（交流电路中的总"阻力"） |
| Rs | Stator Resistance | 定子电阻 |
| ωe | Electrical Angular Velocity | 电角速度 = 2π × 电频率 |
| Ls | Stator Inductance | 定子电感 |
| j | — | 虚数单位（电感的"阻力"和相位有关） |

```
Z = Rs + j × ωe × Ls
|Z| = sqrt(Rs² + (ωe × Ls)²)          sqrt = 开平方根

当 ωe × Ls << Rs 时:  |Z| ≈ Rs       ← 低频退化
当 ωe × Ls >> Rs 时:  |Z| ≈ ωe × Ls  ← 高频正常
```

### DQ 轴电压需求（id=0 策略）

| 符号 | 含义 |
|------|------|
| V_ph | Phase Voltage，相电压需要多大 |
| id | d 轴电流（励磁分量），id=0 意思是"不额外励磁" |
| iq | q 轴电流（转矩分量），产生旋转力的电流 |
| ψm | Flux Linkage（磁链）= Flux，单位为 Wb |
| BEMF | Back EMF = ωe × ψm，反电动势 |

```
V_ph = sqrt( (ωe × Ls × Iq)²  +  (Rs × Iq + ωe × ψm)² )
        └── 电感压降 ──┘     └── 电阻压降 ──┘ └─ BEMF ─┘
```

### 低频下的简化公式

低频时 ωe × Ls ≈ 0（感抗可忽略）：

```
Iq_max ≈ (V_available - ωe × ψm) / Rs
```

### 转矩能否克服负载的判据

| 符号 | 含义 |
|------|------|
| Te | 电磁转矩（电机自己出的力） |
| KT | 转矩常数 = 1.5 × Np × Flux |
| I_LIMIT | 软件设定的电流上限 |
| Tcog | 齿槽转矩（Cogging Torque）— 铁芯槽和磁铁之间的吸力 |
| Tfriction | 静摩擦转矩 — 轴承、装配产生的阻力 |
| b × ωm | 粘滞阻尼 — 和转速成正比的阻力，转越快阻力越大 |
| stall | 堵转 — 通电但不转 |

```
Te = KT × min(Iq_max, I_LIMIT)

Te ≥ 齿槽峰值 + 静摩擦 + 粘滞阻尼   → 能动 ✓
Te < 齿槽峰值 + 静摩擦              → 堵转（stall）✗
```

## 在 V/f 仿真中怎么办

| 问题 | 对策 |
|------|------|
| 低频时 Rs 吃掉大量电压 | V/f 曲线加 **v_boost**，即 v_d = v_boost + Kv × f |
| 齿槽太大卡住转子 | 改用电流闭环（后续模型 x6）：强制给 iq，不受 Rs 和 BEMF 限制 |
| 电流限幅太低 | 调高 `iq_limit` |
| V/f 本身低速就弱 | 有感 FOC（x7/x8）才是解决之道——用编码器精确控制 iq 的相位 |

### 什么是 V/f

**V/f** = Voltage / Frequency 控制。核心公式：

```
v_d = Kv × f + v_boost
```

- Kv × f：电压跟着频率比例增长（速度越高，反电动势越大，需要匹配的电压越高）
- v_boost：低速抬升电压，补偿电阻压降和死区损耗

你的电机 Kv 极小（0.00365 V/Hz），所以哪怕 50Hz 也只要 0.2V 左右。

```
2204 2300KV 电机:

  电气时间常数 τe = 80 μs
    → 电流响应极快，≈ 1 个 PWM 周期
    → 一般电机 τe 是几毫秒，你是它的 1/50

  转矩常数 KT = 0.00357 Nm/A
    → 每安培只出 3.6 毫牛·米，非常"软"
    → 要出 0.1 Nm 需要 28A，远超额定

  电感 Ls = 9 μH
    → 极低电感是高 KV 的代价
    → 对 PWM 频率极度敏感，需要高开关频率

  满速 27000 RPM
    → 需要 3150 Hz 电频率
    → 低速控制反而是硬骨头

V/f 起步经验值:
  从 f = 5~10 Hz 开始，v_d = 0.3~0.5 V
  稳定后逐步升 f
  v_boost ≥ 0.2V（补偿死区和管压降）
```
