# V/f 开环振荡 — 物理根因与调参策略

## 先看你的数据

```
稳态振荡: ±150 rad/s
平均转速: ~8.4 rad/s (80 RPM)
指令转速: 17.9 rad/s (171 RPM, f=20Hz)
振荡频率: ~4-5 Hz (机械), 周期~0.2s
```

**结论：电机在转，但平均速度只有指令值的一半，且以固有频率持续振荡。这不是不稳定，而是 V/f 的天生特性——这个电机没有阻尼。**

---

## 为什么会振荡

### 物理模型

把 PMSM 在 V/f 模式下看作一个**弹簧-质量系统**：

| 类比 | 机械弹簧 | PMSM + V/f |
|------|---------|-----------|
| 弹簧 | F = k × Δx | 电磁转矩 Te ≈ K_elec × (θ_stator - θ_rotor) |
| | 偏离平衡就产生恢复力 | 定子-转子夹角（功角 δ）偏离就产生转矩 |
| 质量 | 转子转动惯量 J | 转子转动惯量 J |
| 阻尼 | 摩擦/粘滞阻力 B | **B_mech + B_elec（电气阻尼）** |

### 阻尼从哪里来

电气阻尼由电机参数决定：

```
B_elec ≈ (1.5 × Np × Flux²) / (Rs × (1 + (ωe×Ls/Rs)²))

你的电机:
  Np = 7, Flux = 0.00034, Rs = 0.112, Ls = 9e-6
  低频时 ωe×Ls/Rs ≈ 0

  B_elec ≈ 1.5 × 7 × (0.00034)² / 0.112
         ≈ 1.5 × 7 × 1.16e-7 / 0.112
         ≈ 1.09e-5  Nm·s/rad
```

| 电机 | Flux (Wb) | Rs (Ω) | B_elec (Nm·s/rad) |
|------|:--:|:--:|:--:|
| **你的 2204** | 0.00034 | 0.112 | **~1e-5** |
| 普通 4310 | 0.00525 | 0.45 | ~3e-4 |
| 230V 伺服 | 0.1 | 1 | ~0.3 |

**你的电气阻尼是一般电机的 1/30、伺服电机的 1/30000。**

### 阻尼不足 = 振荡不衰减

```
阻尼比 ζ = B_total / (2 × sqrt(K_elec × J))

正常电机: ζ = 0.1~0.5  → 振荡几下就收敛
你的电机: ζ ≈ 0.001    → 几乎为零，一旦振荡就停不下来
```

### 为什么一开始就振荡

```
t=0: 定子磁场开始以 20Hz 旋转
     转子静止，功角 δ 瞬间拉到 90°+
     → 产生巨大转矩（冲击电流）
     → 转子猛冲过去
     → 冲过头，δ 变负
     → 反方向拽
     → 来回振荡 ← 阻尼太小收不住
```

## 能调到什么程度

### V/f 固有的极限

V/f 模式下你能控制的只有两个东西：
- **电压幅值**（v_d）：决定电磁力多强
- **旋转频率**（f）：决定磁场转多快

你**不能**控制：
- 电流大小（它由电压+反电动势+阻抗+负载共同决定）
- 功角 δ（转子自己找到的平衡点）
- 阻尼（由电机物理参数决定）

**所以 V/f 注定振荡，只能减小振幅，不能消除。**

### 调参策略

#### 1. 降低振荡振幅：让 v_d 精准匹配

```
最佳 v_d = v_boost + Kv × f

其中:
  v_boost ≈ 死区损耗 + Rs × (齿槽所需最小电流)
          = 0.48 + 0.112 × 3
          ≈ 0.8 V

  Kv ≈ (vbus - v_boost) / f_max_elec
     = (12 - 0.8) / 3150
     = 0.00355 V/Hz

f = 20 Hz:  v_d = 0.8 + 0.00355 × 20 = 0.87 V
f = 50 Hz:  v_d = 0.8 + 0.00355 × 50 = 0.98 V
f = 100 Hz: v_d = 0.8 + 0.00355 × 100 = 1.16 V
```

v_d 太大 → 冲击电流 → 大振幅振荡
v_d 太小 → 转不动
v_d 刚好 → **最小振幅**（但不为 0）

#### 2. 降低起振冲击：加频率斜坡

不要 f 从 0 直接跳到 20Hz。用一个 Ramp 或 Step block 控制 f 从 5 逐步升到 20：

```
f = 5Hz 起步 → 稳定 0.3s → 升到 10Hz → 稳定 → 升到 20Hz
```

每个台阶只跨 5Hz，转子跟得上，冲击就小。

#### 3. 加机械阻尼（仿真的 tricks）

在 PMSM 的 Mechanical 页，给 Rotor damping (B) 加一个值：

```
原来: B ≈ 0 或 1e-6
改成: B = 0.0005
```

这在物理上等价于给转子加了一个**旋转阻尼器**——转越快阻力线性增大。它会吸收振荡能量，让转速收敛。

> 注意：这是仿真作弊手段，实际电机改不了 B。目的在于验证"给够阻尼就能稳"，从而确信根因是阻尼不足。

#### 4. 降低频率：减振

在可以接受的低速运行：

```
f = 5 Hz (43 RPM):
  v_d = 0.8 + 0.00355×5 = 0.82 V

转速低 → 动能小 → 振荡幅值小 → 相对稳定
```

## 实操顺序

| 步骤 | 操作 | 期望 |
|:--:|------|------|
| 1 | PMSM 阻尼 B 改为 **0.0005** | 振荡大幅衰减甚至消失 |
| 2 | 确认阻尼有用后，改回 B=0 | 回到现实 |
| 3 | v_d = **0.87V**，f=20Hz | 振幅比之前的 3V 小 |
| 4 | 微调 v_d ±0.05 找最小振幅点 | |
| 5 | f 从 5→10→15→20Hz 逐步升 | 避免启动冲击 |
| 6 | 接受残余振荡 | 进入 x6 电流闭环 |

## 底线

**V/f 开环 + 9μH 电感 + 2300KV 电机 = 必然振荡。不是你的仿真错了，是控制结构决定的。你调得再好，振幅也只能降低一半左右。要根治振荡，必须上电流闭环——电流闭环直接钳住 iq，不让它因为功角变化而忽大忽小。**
